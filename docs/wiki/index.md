# Wiki

这里对应的是派生知识层。

当前已经直接复制到站点静态文件区：

- `/raw/wiki/repowiki-content/`
- `/raw/wiki/repowiki-knowledge/`

## 它适合做什么

repowiki 更适合承担这些用途：

- 新人快速理解项目
- 横向浏览模块关系
- 用更浓缩的方式查看架构、数据库、API 和部署说明

## 它不适合做什么

- 覆盖 `Lucent/docs` 和 `Luminous/docs` 的当前规则
- 成为部署、运行、生成契约的最终依据
- 代替 owning repo 中随代码更新的文档
