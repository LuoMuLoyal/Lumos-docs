# Current Docs

Current Docs 指的是当前仍然生效、并且应跟随各自项目演进维护的文档。

当前主要来源：

- `../Lucent/docs`
- `../Luminous/docs`

## 用法

- 想找后端运行、部署、契约、数据源文档，先去 `Lucent/docs`。
- 想找前端产品、当前状态、迁移日志、OpenAPI 客户端流程，先去 `Luminous/docs`。
- 这些文档的编辑仍然应在各自仓库完成，不建议在文档站里手工维护副本。
