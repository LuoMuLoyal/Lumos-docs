## AI Chat Phase 1 前端首版

- 新增 `features/ai_chat/`，补齐最小可用的 AI chat 前端模块：domain model、remote datasource、repository、Riverpod controller、聊天页。
- Settings 隐私区新增 `AI 对话` 入口，已登录用户可进入独立 `/assistant` 页面。
- AI chat 页现已接上真实能力探测与真实 SSE 流式输出：
  - 使用 `GET /api/v1/user/ai-chat/capabilities`
  - 使用 `POST /api/v1/user/ai-chat/messages/stream`
  - 明确继续复用 `lib/core/network/lucent_sse.dart`，不走生成 OpenAPI 的伪流式方法
- 聊天页首版可做的事：
  - 展示后端 readiness / tool / context / RAG 状态
  - 开关 AI chat 总开关
  - 开关 health profile / recent records / sleep data / current medicines 四类上下文权限
  - 发送真实消息并渲染 Markdown 助手回复
  - 在未登录、能力拉取失败、后端未就绪、模型未配置时显示明确状态
- 将已停更的 `flutter_markdown` 替换为 `flutter_markdown_plus`。这是原包停更后的延续维护版，现阶段更适合我们这种“只需要稳定 Markdown 聊天气泡”的场景，不必为了暂时用不上的 AI/LaTeX 特性扩大依赖面。
