# 2026-06-28 Luminous 设置页重构

## 变更摘要

将设置页从「平铺分组 + 多处 dialog/行内操作」重构为标准 App 式的一级/二级/三级页面结构（参考百度网盘式设计）。

## 主要改动

### 页面结构

- 设置首页按「账号与安全、通知、隐私、通用、关于」五组重新组织。
- 新增顶部账号入口卡片，保留邮箱/昵称展示。
- 新增子页面：
  - `/settings/notifications` — 通知设置
  - `/settings/notifications/sleep` — 睡眠提醒详情
  - `/settings/ai` — AI 设置
  - `/settings/export` — 数据导出
  - `/settings/help` — 帮助与反馈
  - `/settings/about` — 关于

### 交互规范

- 二态设置项（如数据共享、AI 总结）使用整行可点击的 `Switch`。
- 多状态设置项进入子页（如主题、语言、AI 设置）。
- 睡眠提醒子页采用「总开关 + 子项灰显」模式：总开关关闭时，就寝时间/起床时间置灰且不可点击。
- 未登录用户仍能看到所有设置项，点击需要登录的项时弹出登录提示。

### 组件抽象

新增可复用组件：
