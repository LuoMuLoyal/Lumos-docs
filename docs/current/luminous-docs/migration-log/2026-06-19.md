# Migration Log - 2026-06-19

## Shared Git Hooks For Flutter Repo

- Added shared repo-local git hooks under `.githooks/` for the Flutter repo instead of relying on per-developer manual `.git/hooks` edits.
- Added `tool/install_git_hooks.ps1` so contributors can configure `core.hooksPath` to `.githooks/` with one command after cloning.
- `pre-commit` now runs `flutter gen-l10n`, `dart format --output=none --set-exit-if-changed .`, and `flutter analyze`.
- `pre-push` now reuses `tool/run_daily_checks.ps1` so pushes are blocked if the repo-safe Flutter validation gate fails.

- Added expired-proposal UX guardrails: expired proposals now show an inline warning and the confirm action is disabled before any write flow is reached.
