## 这不是唯一真相源

这个站点的作用很简单：把 Lumos 工作区里分散的文档入口放到一起，降低查找成本。

## 先看哪里

- 如果你要找当前生效的项目文档，先看 [Current](/current/)。
- 如果你要快速理解全局结构、模块关系、数据库和 API 摘要，再看 [Wiki](/wiki/)。
- 如果你是在做历史追溯，再看 [Archive](/archive/)。
